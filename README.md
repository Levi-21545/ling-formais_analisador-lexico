
## Analisador Léxico

Este projeto é um analisador léxico com suporte a múltiplos tokens, desenvolvido para facilitar o reconhecimento e a categorização de elementos léxicos em cadeias de texto. A aplicação possui interface intuitiva e permite visualizar o processo de análise em tempo real.

### Como rodar localmente

1. Instale as dependências:
	```bash
	npm install
	```
2. Inicie o servidor de desenvolvimento:
	```bash
	npm run dev
	```
3. Acesse `http://localhost:5173` no navegador.
